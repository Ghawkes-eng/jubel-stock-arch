How to connect in 3 steps:
1. Go to sheetdb.io and sign in with Google
2. Click Create API and select your Google Sheet
3. Copy the API URL and paste it above

Your sheet needs these column headers in row 1:
Timestamp, Name, SKU, Type, Qty, Live Stock
